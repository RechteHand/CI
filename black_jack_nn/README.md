# Black Jack (Neuronales Netz / RL)

Dieses Projekt beinhaltet ein in C entwickeltes Modell, das basierend auf der eigenen Hand und der Karte des Dealers selbstständig die optimale Blackjack-Strategie erlernt. Es nutzt Ansätze aus dem Bereich Neuronale Netze und Reinforcement Learning.

## Kompilieren und Starten

1. **Kompilieren**: Das Projekt verwendet ein `Makefile`. Führe einfach `make` im Ordner aus.
   ```bash
   make
   ```
2. **Starten**: Führe die erstellte Binärdatei aus (in der Regel `./main` oder ähnlich).
   ```bash
   ./main
   ```
