# Computational Intelligence - HFT Stuttgart

Dieses Repository wurde für das Modul **Computational Intelligence** an der Hochschule für Technik (HFT) Stuttgart erstellt. Es beinhaltet verschiedene Projekte und Implementierungen, die im Rahmen dieses Moduls erarbeitet wurden.

## Projekte aus der Präsentation

Folgende Projekte wurden in der Präsentation angesprochen. Jedes Projekt befindet sich in dem jeweiligen Unterordner:

1. **Genetischer Algorithmus (Inselmodell)**
   - Ordner: `genetic_algorithm_olympic_island`
2. **Drohnen Fußball-Show**
   - Ordner: `drohnen_finale89` (bzw. `drohnen_finale89_ohne_raster`)
3. **F1 AI Racing (Neuroevolution)**
   - Ordner: `f1_rl`
4. **Tetris (Reinforcement Learning)**
   - Ordner: `reinforcement_learning_tetris`
5. **Flappy Bird (Neuroevolution)**
   - Ordner: `flappy_bird`
6. **Black Jack (Neuronales Netz / RL)**
   - Ordner: `black_jack_nn`
7. **Katzen vs. Hunde Klassifikation (CNN)**
   - Ordner: `cat_vs_dog_classification`
